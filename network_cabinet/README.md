## 🗄️ Network Cabinet Overview
The home lab is built around a **wall-mounted network cabinet**, customised for both aesthetics and usability:

- Interior painted **black**
- Exterior finished in **IKEA white**
- **Wheels fitted underneath** for mobility and easier maintenance

---

## 🔌 Patch Panel
- Type: **CAT5e Patch Panel**
- All structured cabling terminates here

### Configuration Standard
- **T568B** wiring standard used

---

## 🔁 Network Switch

### Original Switch
- 24-Port Gigabit Switch (10/100/1000)

### Current Switch
- 48-Port Switch
- Speeds:
  - 10/100 (majority of ports)
  - 2x Gigabit uplink ports

### Configuration
- Standard devices → 10/100 ports  
- High-priority devices → Gigabit ports
