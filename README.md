# First Homelab Documentation

Welcome to my **first home lab** documentation! This repository is a place to record the setup, configuration, and lessons learned from building and running my home lab environment.

## Project Overview

This project documents my current home lab and network setup before a planned full rebuild. The goal is to keep a clear record of:

- Hardware and devices used
- Network structure and topology
- Configuration details for servers and services
- Lessons learned and improvements for future iterations

## Current Scope

At this stage, the documentation focuses on the basics:

- Inventory of devices and components
- Network layout
- Key services currently running
- Notes on what works well and what could be improved

## Future Plans

Over time, this repository will expand to include:

- Step-by-step guides for configuration and setup
- Troubleshooting notes and tips
- Diagrams of network and service architecture
- Automation scripts and tools used in the lab

## Contribution

This is a personal project, but contributions, suggestions, and feedback are welcome via GitHub issues or pull requests.
